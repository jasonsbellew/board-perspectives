# Board Review Prompts

Use these prompts with Board Perspectives to pressure-test business decisions before spending money or committing resources.

## Full Board Review

```text
Conduct a full board review.

Decision:
[Describe the decision]

Context:
[Current situation]

Options:
1.
2.
3.

Constraints:
Budget:
Timeline:
Risks:

Goal:
What would the board recommend and why?
```

## Product or Service Launch

```text
I’m considering launching [product/service].

Startup cost:
[$ amount]

Audience:
[Who it is for]

Current evidence:
[Pre-sales, survey results, customer interviews, waitlist, usage data, or other proof]

Run a full board review and tell me whether to proceed, test small first, delay, or kill it.
```

## Software Build

```text
Pressure-test this idea:

I want to spend [$ amount] building a software tool because [reason].

Current evidence:
[Pre-sales, signed customers, usage data, customer interviews, or feedback]

Should I build it, test first, delay, or kill it?
```

## Capital Allocation

```text
Conduct a full board review.

I have [$ amount] available and need to decide where to allocate it.

Options:
1. Invest in marketing
2. Hire someone
3. Improve operations
4. Pay down debt
5. Keep cash reserves

Context:
[Describe the business situation, goals, and risks]

Goal:
Which option creates the best risk-adjusted return?
```

## Hiring Decision

```text
Conduct a hiring review.

Role:
[Position]

Candidate or context:
[Candidate details or hiring need]

Budget:
[Compensation range]

Current bottleneck:
[What problem this hire is supposed to solve]

Goal:
Should I hire now, hire with conditions, run a trial period, delay, or decline?
```
